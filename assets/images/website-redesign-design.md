# “云衍元极”网站视觉改版说明

## 1. 设计定位

新版网站定位为“个人研究操作系统”：兼顾学术可信度、技术探索感与个人表达，不做过度炫目的赛博朋克，也避免常见的模板化个人博客观感。

核心关键词：

- 克制科技感：深海蓝背景、精细网格、轨道线与微弱粒子。
- 学术可信度：清晰的信息层级、论文与项目优先、内容易扫描。
- 品牌辨识度：沿用红蓝双螺旋标志，并把“无限、演化、连接”转化为页面中的轨道视觉。
- 人格化表达：角色形象作为“数字研究伙伴”出现，为严肃内容增加记忆点，但不喧宾夺主。

## 2. 三份素材的使用方式

### `logo.svg`

- 作为导航栏主标志、按钮图形和页面加载动效的核心。
- 红色表示探索与行动，蓝色表示理性与结构。
- 标志中的双眼/双核概念延伸为首屏的两条数据轨道和卡片边缘光。

### `white-logo.png`

- 不直接作为大幅横幅使用，避免书法字体与科技界面产生割裂。
- 将“云衍元极”作为首屏或页脚的低透明度艺术签名。
- 标语“始于一元，衍至无极”升级为网站的主品牌语。

### `light_figure3-4.png`

- 放在首页首屏右侧，定位为“研究者数字分身/智能伙伴”。
- 角色周围加入红蓝轨道、数据节点和研究方向标签。
- 保留白色服装与原有标志，使角色自然融入深色背景。

## 3. 首页结构

1. **悬浮导航栏**
   - 左侧为标志与“云衍元极”。
   - 中间保留：首页、关于我、博客文章、学术、生活、联系方式。
   - 右侧为中英文与明暗主题切换。

2. **品牌首屏**
   - 左侧：研究身份、主品牌语、简短介绍与两个主要入口。
   - 右侧：角色形象、红蓝轨道和研究方向状态卡。
   - 背景只使用低对比网格和粒子，保证文字阅读性。

3. **研究状态条**
   - 用短标签展示当前身份、研究兴趣和最新成果。
   - 比传统“个人简介大段文字”更适合快速浏览。

4. **三大内容入口**
   - 学术研究：论文、项目、学术主页。
   - 技术博客：机器学习、大模型、强化学习、优化。
   - 生活札记：复盘、观察与成长记录。

5. **精选成果**
   - 首页直接展示最新论文 `HeLoFusion`。
   - 提供 arXiv、PDF 等明确操作入口。

## 4. 视觉系统

### 色彩

- 页面底色：`#07101F`
- 深层面板：`#0C1728`
- 品牌蓝：`#3C8DFF`
- 品牌红：`#E33B43`
- 主文字：`#F3F7FC`
- 次级文字：`#8FA3BC`

红色只用于关键动作和焦点，蓝色负责导航、结构和环境光。

### 字体

- 中文标题：思源宋体 / Noto Serif SC，保留文化气质。
- 中文正文：思源黑体 / Noto Sans SC。
- 英文标签与数据：Inter / Space Grotesk / 等宽字体。

### 组件

- 卡片采用低透明深色玻璃，而不是大面积白色毛玻璃。
- 圆角控制在 16–24px，边框使用 1px 半透明蓝灰色。
- 悬停时只做 2–4px 位移、边框增亮和局部光晕。
- 背景标志不再铺满全页，避免干扰内容。

## 5. 动效建议

- 首屏轨道以 18–24 秒缓慢旋转，粒子沿轨道移动。
- 页面滚动时，内容卡片分层淡入并轻微上移。
- 导航栏滚动后缩小高度并提高背景不透明度。
- 角色只做非常轻微的悬浮，不使用频繁摇摆或眨眼。
- 支持 `prefers-reduced-motion`，照顾减少动画的系统设置。

## 6. 响应式策略

- 桌面端：首屏左右分栏，角色占宽约 38%。
- 平板端：首屏保持分栏，但减少装饰节点。
- 手机端：文字在上、角色在下；导航折叠为菜单；内容卡片改为单列。
- 手机端隐藏大部分轨道和粒子，优先保证加载速度与阅读体验。

## 7. 实施优先级

1. 重构全局色彩变量、导航栏和首页首屏。
2. 统一卡片、按钮、标签和论文项目组件。
3. 将同一视觉系统扩展到博客、学术、生活和文章详情页。
4. 最后加入轨道、粒子和页面过渡动效，并进行移动端性能优化。
